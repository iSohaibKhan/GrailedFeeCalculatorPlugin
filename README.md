<h1>Grailed Fee Calculator Tool (WordPress Plugin)</h1>
<p>This is a shortcode plugin created for WordPress websites using <b>PHP</b>, <b>HTML</b>, <b>CSS</b>, and <b>JavaScript</b> and this tool is made to calcualte your estimated profits on <b>Grailed Marketplace Platform</b> in USD, CAD, and GBP, this tool fetches the typography and colors from the theme dynamically to match the design with the website it is on and it is mobile responsive too.</p>
<h3>Instructions:</h3>
<ul>
  <li>Download the plugin's zip file from this repository.</li>
  <li>Upload it to your WordPress website and activate the plugin.</li>
  <li>To display it on the front-end of your website use the shortcode <b>[grailed_fee_calculator]</b></li>
</ul>
